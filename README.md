
# 🎬 Cinephile Landing Page

Landing page moderna creada con **React + Vite + TailwindCSS**, diseñada como proyecto de práctica para una plataforma de cine/series.

[Github pages](https://cinephile-team5.github.io/cinephile-e5/)

- ⚛️ React 19.2.6
- ⚡ Vite 8.0
- 🎨 Tailwind CSS 4.3
